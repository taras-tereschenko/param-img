# Param Img

**Make your photos Instagram Story ready in seconds.**

[**Try it now**](https://param-img.vercel.app) | No signup required

---

## Why Param Img?

Uploading a 4:3 photo to Instagram Stories? Get ready for awkward stretching, random cropping, and no way to center your image properly. Instagram's built-in editor just doesn't cut it.

Param Img gives you full control. Choose beautiful backgrounds, center your photos perfectly, and export Story-ready images in seconds.

### Local-First & Private

**Your photos never leave your device.** Everything happens right in your browser:

- No uploads to external servers
- No cloud storage
- No account required
- Works completely offline

Your images stay yours.

### Key Features

- **Perfect 9:21 ratio** - Optimized for Instagram Stories
- **Beautiful backgrounds** - Blur, ambient glow, or solid colors
- **Proper centering** - Your photo, exactly where you want it
- **Batch processing** - Handle multiple images at once
- **Installable** - Add to home screen, works like a native app
- **Share from gallery** - Send photos directly to Param Img from any app
- **Share to anywhere** - Export to Instagram, Messages, or any app

## Features

### Background Modes

| Mode        | Description                                   |
| ----------- | --------------------------------------------- |
| **Blur**    | Your image, beautifully blurred behind itself |
| **Ambient** | YouTube-style cinematic glow effect           |
| **Solid**   | Clean black, white, or any custom color       |

### Image Controls

- **Scale** - Resize from 50% to 100%
- **Border Radius** - Add rounded corners (None, S, M, L)

### Export Options

- **Batch Download** - Export all images with one click
- **Native Share** - Share directly to any app
- **Lossless PNG** - No quality loss, perfect output

### Offline-First PWA

- **Installable** - Add to your home screen like a native app
- **Works offline** - No internet? No problem
- **Share Target** - Select photos in your gallery, tap Share, choose Param Img
- **Auto-Save** - Images persist locally between sessions

---

## How to Use

1. **Drop** images or tap to select
2. **Style** - Choose background mode and adjust settings
3. **Export** - Download or share to Instagram

## Tech Stack

Built with modern web technologies:

- **React 19** + **TypeScript**
- **TanStack Start** - Full-stack React framework
- **Tailwind CSS 4** - Utility-first styling
- **Vite** + **Bun** - Lightning-fast builds
- **Web Workers** - Smooth UI during processing
- **IndexedDB** - Persistent local storage

## Development

```bash
# Install dependencies
bun install

# Start dev server
bun run dev

# Build for production
bun run build

# Run checks
bun run check
```
